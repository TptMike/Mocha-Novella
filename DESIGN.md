## Design System

### Color Palette
```
Primary Blue: #3b82f6    (Tech-forward, trustworthy)
Purple Accent: #8b5cf6   (Creative, unique)
Amber Highlight: #f59e0b (Warm, attention-grabbing)
Deep Slate: #1e293b      (Professional text)
```

### Typography
- **Display Font**: Georgia (serif) - for headings, elegant and readable
- **Body Font**: System fonts (Inter/Segoe UI) - fast, clean, modern
- **Monospace**: SF Mono - for code, technical content

### Spacing Scale
Uses a consistent 8px base unit system:
- xs: 8px
- sm: 12px  
- md: 16px
- lg: 24px
- xl: 32px
- 2xl: 48px
- 3xl: 64px
- 4xl: 96px

### Shadow System
- **sm**: Subtle depth
- **md**: Card elevation
- **lg**: Hover states
- **xl**: Modal/overlay depth
