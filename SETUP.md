# Quick Setup Guide

## Activating Your Theme

1. Log into your WordPress admin panel
2. Go to **Appearance > Themes**
3. Find "Mike Personal" in your themes list
4. Click "Activate"

## First Steps After Activation

### 1. Set Up Your Menu
- Go to **Appearance > Menus**
- Create a new menu (or edit an existing one)
- Add pages like: Home, About, Books, Projects, Blog
- Assign it to "Primary Menu" location
- Save

### 2. Add Your Logo
- Go to **Appearance > Customize > Site Identity**
- Click "Select Logo" and upload your logo
- Adjust size if needed
- Publish changes

### 3. Configure Social Media Links
- Go to **Appearance > Customize > Social Media Links**
- Add your social media URLs:
  - Twitter/X
  - Facebook
  - Instagram
  - LinkedIn
  - GitHub
  - Email (just enter your email address)
- Publish changes

### 4. Customize Footer Text (Optional)
- Go to **Appearance > Customize > Theme Options**
- Add custom footer text if desired
- Or leave blank for default copyright notice

### 5. Set Up Your Homepage
You have two options:

**Option A: Use Latest Posts (Blog)**
- Go to **Settings > Reading**
- Select "Your latest posts"

**Option B: Use a Static Page**
- Create a new page called "Home"
- Create another page called "Blog"
- Go to **Settings > Reading**
- Select "A static page"
- Choose "Home" as homepage
- Choose "Blog" as posts page

### 6. Add Widgets (Optional)
- Go to **Appearance > Widgets**
- Add widgets to "Sidebar" for blog pages
- Add widgets to "Footer Widget Area" for footer

## Recommended Widgets for Authors

### Sidebar Widgets:
- **Recent Posts** - Show your latest blog posts
- **Categories** - Help readers find topics
- **Tag Cloud** - Show popular tags
- **Text Widget** - Add author bio or newsletter signup

### Footer Widgets:
- **Text Widget** - Newsletter signup
- **Recent Posts** - Latest updates
- **Social Links** (if you have a plugin)

## Creating Your First Post

1. Go to **Posts > Add New**
2. Write your post
3. **Set Featured Image** - Important for visual appeal
4. Add categories and tags
5. Publish!

## Tips for Book Authors

1. **Create a "Books" Category**: Use this for all book-related posts
2. **Use Featured Images**: Always add compelling featured images
3. **Reading Lists**: Create pages for "Reading Lists" or "Recommendations"
4. **Book Updates**: Post about your writing progress, cover reveals, etc.

## Tips for Project Showcases

1. **Project Categories**: Create categories for different project types
2. **Image Galleries**: Use WordPress galleries in posts
3. **Sticky Posts**: Use sticky posts to highlight featured projects
4. **Portfolio Pages**: Create dedicated pages for your portfolio
